## What is ROPE?

ROPE is a simple, beautiful Bible journaling app that guides you through four daily steps:

- **R** — **Revelation**: Look up a Bible verse that speaks to you today
- **O** — **Observation**: Write what you notice in the passage
- **P** — **Prayer**: Respond to God in prayer
- **E** — **Execution**: Commit to one way you'll live it out

The next day, you check in on your commitment. Over time, ROPE helps you see patterns in your spiritual growth — which books you return to, how consistently you follow through, and what God might be saying through it all.

## Features

- Guided daily journaling through the ROPE method
- Automatic Bible verse lookup (powered by bible-api.com)
- Voice-to-text input for hands-free journaling
- Execution check-ins to build follow-through
- Insights dashboard with streaks, book frequency, and growth patterns
- Works offline — all data stays in your browser
- Clean, warm, distraction-free design

## Use It

ROPE is live at **[rope-app-lake.vercel.app](https://rope-app-lake.vercel.app)**. No account needed to start — just sign up and begin journaling.

## For Developers

Want to run it locally?

```bash
git clone https://github.com/your-username/rope-app.git
cd rope-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and you're in.
